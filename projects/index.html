zenBooking

The project is an online travel and accommodation booking platform that enables users to search, compare, and book hotels, vacation rentals, and travel services. It features advanced search filters, detailed property listings, secure payment processing, and user accounts for managing bookings. Property owners can list and manage their offerings, while users can leave reviews, ensuring a seamless and trustworthy experience for travelers and hosts alike.

scatch

Scatch is an online bag store with a strong focus on backend development. Built using Node.js, Express, and MongoDB, it features robust APIs for product management, user authentication, and order processing. The frontend, powered by EJS, provides a simple and responsive interface for browsing bags, managing carts, and placing orders. This project emphasizes scalability, security, and efficient data handling for a seamless shopping experience.
